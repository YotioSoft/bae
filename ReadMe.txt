--------------------------------------------------------------------------------

				  イ○スタ蝿

--------------------------------------------------------------------------------

1.はじめに
　この度は、「イ○スタ蝿」をダウンロードしていただき、ありがとうございます。
　本書では、このゲームの操作方法について簡単にご説明いたします。

2.概要
　あなたはスマートフォンで写真を撮ろうとしています。
　しかし、あなたの周りには\nイ○スタバエがたくさんいます。
　このイ○スタバエがなるべく写らないように、
　電池の残量に注意して\n撮影ボタンを押しましょう。

3.操作方法
　1: イ○スタ蝿.exeを起動します。
　2: 「はじめる」を押します。
　3:　蝿が減るタイミングを見計らって下の撮影ボタン（丸いボタン）を押します。
　4:　結果が表示されます。
